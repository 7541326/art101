Lab 10 - Javascript for the Web

Challenge

The challenge for this lab was to create a button with an attached function, that will append a random selection of text to a selected div.

Problems

I encountered some troubles with extra task stuff, such as the input form and the timestamp things, but I was able to figure them out by Googling them.

Reflection

While I was able to get the bonus task of the left/right aligned text to mimic messages, I decided against using it on the final page. I like the timestamp I put, as well as the fact that I could make it so that you can't "send" an empty message to the output div. Also able to press enter to send while the input is selected, which I think is nice. I also changed the random text generator, so that I could have it put up a randomly selected pre-defined quote, instead of a randomly sliced selection of text.
